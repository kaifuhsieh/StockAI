# 飆股管家 - 專業股票分析 APP 配置方案

我已經為您開發了一個高品質、深色系的專業股票分析儀表板。這個介面是基於您提供的照片資訊所設計的，包含了所有關鍵的技術指標與數據。

## ✨ 設計亮点

- **專業深色系 (Premium Dark Mode)**: 使用了深邃的暗色調背景，搭配玻璃擬態 (Glassmorphism) 效果，營造出如 Bloomberg 或高階交易軟體般的專業感。
- **在地化股市色彩**: 採用台灣股市慣用的「紅漲綠跌」配色方案，讓資訊判讀更直覺。
- **響應式佈局 (RWD)**: 專為手機端（如 iPhone 17 Pro）優化，包含底部導航欄與移動端標題，提供原生 APP 般的操作體驗。
- **高階排版**: 使用了 `Inter` 與 `Noto Sans TC` 字體，確保高品質的文字渲染。

## 📁 檔案說明

- **index.html**: 應用程式的主體結構。
- **style.css**: 所有的視覺設計與動畫效果。
- **script.js**: 負責處理個股資料渲染與邏輯。

---

您可以直接在瀏覽器中開啟 `index.html` 來查看動態效果！
